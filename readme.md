## CLI приложение для обработки csv-файлов с метриками видео

Данное CLI приложение собирает данные из указанных файлов типа CSV и формирует отчет в заданном формате отчета. 

### Применение
Запуск приложения возможен двумя способами: 
- командой запуска (по примеру) с явным указанием файлов и типа отчета (python3 main.py --files stats1.csv stats2.csv --report clickbait)
- через файл run, он собирает все файлы в папке FILES и уточняет название отчета.  (python3 run.py)
